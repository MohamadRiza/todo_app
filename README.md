# To-Do App

A To-Do application built with:
- **Frontend**: React, Vite, Tailwind CSS
- **Backend**: Express.js, Node.js, PostgreSQL

## Features
- Add, and delete to-do items.
- Responsive design with a clean UI.
- RESTful API for CRUD operations.
